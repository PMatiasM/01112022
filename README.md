## Getting Started

### Prerequisites

* npm

```sh
npm install npm@latest -g
```

### Installation

1. Clone the repo

```sh
git clone https://github.com/PMatiasM/our-map.git
```

2. Install NPM packages

```sh
npm install
```

3. Enter API URL in `src/services/api.js`

```JS
const baseURL = "your.api.url";
```

4. Have fun!
